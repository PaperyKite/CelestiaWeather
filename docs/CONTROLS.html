地图控件开发规范 ​

1. 布局约定 ​

  • 右侧：所有控制类插件（底图切换、主题/样式控制、图例、图层列表等）统一在右侧自上而下排列，与主操作区分离。
  • 左下角：状态/信息类内容（层级、鼠标实时经纬度、比例尺等）放置在软件左下角，使用主题变量以适配亮/暗色。

2. 控件分类 ​

类型位置说明
控制类右侧底图切换、主题切换、图例、图层控制等，可点击、可展开。
状态类左下角仅展示信息：缩放层级、鼠标经纬度、比例尺等，随主题样式。

3. 样式与主题 ​

  • 控件样式统一使用 主题样式表（theme.css）中的 CSS 变量，便于亮/暗色切换与整体一致。
  • 新增控件时使用变量：--CelestiaWeather-bg、--CelestiaWeather-color、--CelestiaWeather-border、--CelestiaWeather-shadow 等，避免硬编码色值。

4. 交互与穿透 ​

  • UI 层容器保持 点击穿透（pointer-events: none），不阻挡地图操作。
  • 每个控件的根节点单独设置 pointer-events: auto，保证本控件可点击，其余区域事件落到地图。

5. 新增控件步骤 ​

  1. 在 src/vue/components/ 下新增组件，控制类放在右侧容器内，状态类放在左下角容器内。
  2. 在 App.vue 中按类型放入对应区域（右侧列 / 左下角）。
  3. 在 MapClass 的 options 中增加对应开关（如 showXxx），并传入 Vue。
  4. 样式引用主题变量，见 src/styles/theme.css。
